# Portfólio - Nicolas Abatti

Portfólio pessoal desenvolvido para apresentar minha trajetória, experiências, projetos e habilidades na área de tecnologia.

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- Git
- GitHub Pages

## Funcionalidades

- Layout responsivo para desktop e dispositivos móveis;
- Navegação por menu com âncoras;
- Seções de experiência profissional, formação acadêmica, habilidades e projetos;
- Galeria de fotos das competições de programação;
- Links para GitHub, LinkedIn e contato por e-mail.
